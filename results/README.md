# Results and Evaluation

This folder contains performance metrics, test results, and visual
evidence for the **AI-Powered Smart Agriculture System**.

## Contents

### model_performance/
Contains results related to the plant disease detection model:
- Accuracy and loss graphs
- Confusion matrix
- Model evaluation metrics

### irrigation_results/
Contains results related to the automated irrigation system:
- Soil moisture threshold tests
- Pump ON/OFF validation
- Water consumption comparison

### screenshots and results/
Screenshots of:
- Streamlit disease prediction interface
- IoT web dashboard
- System in operation

## Summary
- Plant disease detection accuracy: ~94.4%
- Water usage reduction: ~28%
- Automated irrigation logic validated through test cases
