## Overview
This project is an AI and IoT based smart agriculture system developed to improve
crop health and optimize water usage through automation and intelligent decision-making.

The system integrates:
- Plant disease detection using a CNN (MobileNetV2)
- Automated irrigation using soil moisture sensing and Arduino
- A real-time web dashboard for monitoring and control
